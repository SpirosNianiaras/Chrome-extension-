# ⚡ AI Tab Companion - Quick Start Guide

## 🚀 5-Λεπτη Εγκατάσταση

### 1. Φόρτωση Extension (2 λεπτά)
```bash
# 1. Άνοιξε Chrome
# 2. Πήγαινε στο chrome://extensions/
# 3. Ενεργοποίησε "Developer mode"
# 4. Κάνε κλικ "Load unpacked"
# 5. Διάλεξε τον φάκελο extension/
```

### 2. Πρώτη Χρήση (3 λεπτά)
```bash
# 1. Άνοιξε 5-10 tabs με διαφορετικό περιεχόμενο
# 2. Κάνε κλικ στο extension icon
# 3. Κάνε κλικ "Σκάναρε Tabs"
# 4. Περίμενε 10-30 δευτερόλεπτα
# 5. Δες τα AI groups και συνοψίσεις!
```

---

## 🎯 Βασικές Λειτουργίες

### Σκάναρισμα Tabs
- Κάνε κλικ στο extension icon
- Κάνε κλικ "Σκάναρε Tabs"
- Περίμενε την AI ανάλυση

### Ομαδοποίηση
- Τα tabs ομαδοποιούνται αυτόματα
- Κάθε ομάδα έχει περίληψη
- Κάνε κλικ για να δεις τα tabs

### Κλείσιμο Tabs
- Επιλέγεις tabs με checkboxes
- Κάνε κλικ "Κλείσε Επιλεγμένα"
- Τα tabs κλείνουν αυτόματα

### Εξαγωγή Περίληψης
- Κάνε κλικ "Εξαγωγή Περίληψης"
- Δημιουργείται JSON file
- Περιέχει όλες τις πληροφορίες

---

## ⌨️ Keyboard Shortcuts

- **Ctrl/Cmd + Enter**: Σκάναρε tabs
- **Escape**: Κλείσε popup

---

## 🔧 Troubleshooting

### Extension δεν φορτώνει
- Ενημέρωσε το Chrome
- Επαναλάβε την εγκατάσταση

### AI δεν λειτουργεί
- Το extension χρησιμοποιεί fallback
- Λειτουργεί με domain-based grouping

### Αργή επεξεργασία
- Περίμενε λίγο περισσότερο
- Κλείσε περιττά tabs
